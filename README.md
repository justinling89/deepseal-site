## About DeepSeal

DeepSeal is a local-first encrypted note-taking app with a built-in password manager. Key features:

- **Envelope Encryption** — DEK/KEK dual-layer key hierarchy, one key per note
- **Zero-Trust Architecture** — fully offline, data never leaves your device
- **Paragraph-Level Encryption** — mix public and encrypted content within a single note
- **Secure Sharing** — password-protected, read-once, auto-expiring share links
