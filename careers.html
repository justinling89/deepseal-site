DeepSeal 致力于打造最安全的本地加密笔记应用。我们是一支小而精的团队，重视代码质量、安全性和用户体验。如果你对密码学、隐私保护或构建高性能桌面应用充满热情，我们期待与你共事。

Rust 后端工程师

负责 DeepSeal 核心加密系统的开发与维护，以及 Tauri 桌面框架的 Rust 侧集成。

  • 3 年以上 Rust 开发经验，熟悉所有权、生命周期和异步编程
  • 了解密码学基础，有 AES-GCM、PBKDF2 等实现经验
  • 熟悉 Tauri v2 框架及 Rust/JavaScript 桥接机制
  • 有 SQLite 或其他嵌入式数据库经验
  • 跨平台（macOS / Windows / Linux）构建经验优先

前端工程师

负责 DeepSeal 客户端界面开发，基于 React 和 TypeScript 构建富文本编辑器及交互体验。

  • 3 年以上 React / TypeScript 开发经验
  • 有 ProseMirror、TipTap 或 Slate 等富文本编辑器开发经验
  • 熟悉 Tauri WebView 环境下的前端性能优化
  • 注重 UI 细节，对设计稿还原度有高要求
  • 有桌面端应用开发经验优先

安全研究员

负责 DeepSeal 加密方案的安全审计、渗透测试和漏洞分析。

  • 深入了解主流加密算法及其实现中的常见漏洞
  • 有密码学协议审计或安全评估经验
  • 熟悉桌面应用渗透测试方法（逆向分析、内存取证等）
  • 有 CVE 编号或漏洞报告经验优先
  • 良好的安全报告撰写能力
